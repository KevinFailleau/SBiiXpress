# SBiiXpress

Première version de l'application SBiiXpress, comporte de nombreux bugs ainsi que des soucis d'optimisation.

Cette application comprends plusieurs modules, tels que "CleanerXpress" qui permet de faire un nettoyage du PC, elle comprends également "FontXpress" qui permet de saisir un texte et de choisir une police d'écriture en fonction de celles qui sont installées.
Un launcher est également disponible pour lier les modules et également pour mettre a disposition des raccourcis vers le site SBiiXpress.
